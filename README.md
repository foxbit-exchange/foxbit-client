# Client API for FoxBit (Brazilian) Exchange

## How to install
### From npm registry
``
npm i foxbit-client
``

## Direct from github repository

``
npm i thiagoburgo/foxbit-client
``

or  

``
npm i git+https://github.com/thiagoburgo/foxbit-client.git
``
